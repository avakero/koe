# プライバシーポリシー — Koe

最終更新日: 2025年

## 概要

Koe は日本語音声入力アプリです。ユーザーのプライバシーを尊重し、可能な限りデータをローカルで処理します。

## 収集するデータ

Koe は**いかなるデータも開発者に送信しません**。

## データの処理

### 音声データ
- マイクから録音された音声は、**端末内のみ**で処理されます
- 録音データは一時ファイル（WAV）として保存され、文字起こし完了後に自動削除されます
- 音声データが外部サーバーに送信されることはありません

### 文字起こし（Whisper.cpp）
- 音声認識は Whisper.cpp を使用し、**完全にローカル**で実行されます
- インターネット接続は不要です

### AI テキスト整形（Gemini API — オプション）
- この機能は**任意**であり、ユーザーが自身の API キーを設定した場合のみ有効になります
- 有効にした場合、**文字起こしされたテキストのみ**が Google の Gemini API に送信されます（音声データは送信されません）
- 送信されたテキストは Google の[利用規約](https://ai.google.dev/terms)に従って処理されます
- この機能を使用しない場合、一切のデータが外部に送信されることはありません

### API キーの保存
- Gemini API キーはユーザーの端末内（アプリのデータディレクトリ）にのみ保存されます
- API キーが開発者や第三者に送信されることはありません

## データの保存場所

| データ | 保存場所 | 備考 |
|--------|----------|------|
| 設定情報（API キー、ショートカット、モデル選択） | OS のアプリデータディレクトリ | 平文の JSON ファイル |
| Whisper モデル | OS のアプリデータディレクトリ | 初回ダウンロード時のみ通信 |
| 録音データ（WAV） | OS の一時ディレクトリ | 文字起こし後に自動削除 |

### OS 別の保存先
- **Windows**: `%APPDATA%\koe\`
- **macOS**: `~/Library/Application Support/koe/`
- **Linux**: `~/.config/koe/`

## ネットワーク通信

Koe が外部と通信するのは以下の場合のみです：

1. **初回セットアップ時**: Whisper モデルおよび whisper-cli バイナリのダウンロード（Hugging Face / GitHub）
2. **AI 整形使用時**: 文字起こしテキストの Gemini API への送信（ユーザーが有効にした場合のみ）
