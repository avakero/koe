# サードパーティライセンス — Voxtro

Voxtro は以下のオープンソースソフトウェアを使用しています。

## whisper.cpp

- **用途**: 音声認識エンジン（バイナリをランタイムでダウンロード）
- **ライセンス**: MIT License
- **著作者**: Georgi Gerganov
- **リポジトリ**: https://github.com/ggerganov/whisper.cpp

## Tauri

- **用途**: デスクトップアプリケーションフレームワーク
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/tauri-apps/tauri

## React

- **用途**: フロントエンド UI フレームワーク
- **ライセンス**: MIT License
- **著作者**: Meta Platforms, Inc.
- **リポジトリ**: https://github.com/facebook/react

## cpal

- **用途**: クロスプラットフォーム音声キャプチャ
- **ライセンス**: Apache License 2.0
- **リポジトリ**: https://github.com/RustAudio/cpal

## hound

- **用途**: WAV ファイルの読み書き
- **ライセンス**: Apache License 2.0
- **リポジトリ**: https://github.com/ruuda/hound

## arboard

- **用途**: クリップボード操作
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/1Password/arboard

## enigo

- **用途**: キーボード入力シミュレーション
- **ライセンス**: MIT License
- **リポジトリ**: https://github.com/enigo-rs/enigo

## reqwest

- **用途**: HTTP クライアント（モデル・バイナリのダウンロード）
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/seanmonstar/reqwest

## Vite

- **用途**: フロントエンドビルドツール
- **ライセンス**: MIT License
- **リポジトリ**: https://github.com/vitejs/vite

## zip

- **用途**: ZIP ファイルの展開（whisper-cli バイナリ）
- **ライセンス**: MIT License
- **リポジトリ**: https://github.com/zip-rs/zip2

## dirs

- **用途**: OS 標準パス解決（app_data_dir 等）
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/dirs-dev/dirs-rs

## futures-util

- **用途**: 非同期ストリーム処理（ダウンロード進捗）
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/rust-lang/futures-rs

## windows-sys

- **用途**: Windows API 呼び出し（フローティングウィンドウ制御）
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/microsoft/windows-rs

## tauri-plugin-updater

- **用途**: アプリ自動更新
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/tauri-apps/plugins-workspace

## tauri-plugin-process

- **用途**: プロセス再起動（更新後の自動再起動）
- **ライセンス**: MIT License / Apache License 2.0
- **リポジトリ**: https://github.com/tauri-apps/plugins-workspace

## OpenAI Whisper モデル (ggml)

- **用途**: 音声認識モデルファイル（ランタイムでダウンロード）
- **ライセンス**: MIT License
- **著作者**: OpenAI
- **リポジトリ**: https://github.com/openai/whisper
